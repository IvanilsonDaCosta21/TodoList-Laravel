TodoList Laravel

Este é um projeto desenvolvido em Laravel 12 que implementa um sistema de gerenciamento de tarefas (TodoList).
O objetivo é criar um sistema funcional, seguro e com interface limpa utilizando Bootstrap 5.

Funcionalidades

- Criar, listar, editar e deletar tarefas (CRUD completo)

- Interface responsiva com Bootstrap

- Badges para indicar status das tarefas

- Layout limpo e fácil de usar

- Ícones com Font Awesome

Tecnologias utilizadas

- Laravel 12

- PHP 8.2.21

- MySQL 

- Bootstrap 5

- Font Awesome

1 Instalação
Clone o repositório:

git clone (https://github.com/IvanilsonDaCosta21/TodoList-Laravel.git)


2 Entre na pasta do projeto:

cd todo-list-laravel


3 Instale as dependências:

composer install
npm install
npm run dev


4 Copie o arquivo .env.example para .env e configure seu banco de dados:

cp .env.example .env


5 Rode as migrations:

php artisan migrate


6 Inicie o servidor:

php artisan serve

Uso

- Acesse http://localhost:8000 para testar o aplicativo.

- Crie, edite ou delete tarefas usando a interface web.
